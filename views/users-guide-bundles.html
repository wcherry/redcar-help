Installing new bundles

Not all features of bundles are supported.

Currently supported features include snippets and themes.

Redcar is installed under the ruby-gems directory. Depending on your platform and how you installed Redcar (all users or single user) the default directory may be different.

Redcar should not be running to complete this installation.

Open a command shell and go to the ~/.redcar/cache directory1.
Delete the file textmate_bundles.cache.
Copy the tmbundle file to the ~/.redcar/textmate/Bundles
(Bundles can also be copied to /usr/lib/ruby/gems/ruby-version/gems/redcar-redcar-version/textmate/Bundles)
.

Restart Redcar and it will rebuild the bundles cache file. This could take a while.

1 The tidal character (~) refers to the home directory on Unix. On Windows XP™ this refers C:\Document and Settings\_username_\ and on Windows Vista™ this refers to C:\Users\_username_.
